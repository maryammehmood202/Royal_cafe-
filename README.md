README ......# Royal_cafe-

A PHP and MySQL based web application designed to streamline cafe operations and enhance the customer ordering experience. The system allows customers to browse menu items, place orders, manage carts, and provide feedback, while administrators can efficiently manage cafe operations through a dedicated dashboard.

---

##  Features

* User Registration & Login
* Customer Profile Management
* Menu Browsing
* Add to Cart Functionality
* Online Order Placement
* Checkout System
* Order History Tracking
* Customer Feedback & Complaints
* Contact Us Page
* Admin Dashboard
* Database Integration with MySQL
* Responsive and User-Friendly Interface

---

## 🛠 Technologies Used

* PHP
* MySQL
* HTML5
* CSS3
* JavaScript

## 📂 Project Structure

* `index.php` – Home Page
* `menu.php` – Display Menu Items
* `cart.php` – Shopping Cart
* `checkout.php` – Order Checkout
* `order.php` – Order Processing
* `login.php` – User Login
* `register.php` – User Registration
* `dashboard.php` – Admin Dashboard
* `feedback.php` – Customer Feedback
* `complaint.php` – Complaint Management
* `contact.php` – Contact Page
* `style.css` – Styling File

##  Installation Guide

1. Clone the repository:

   ```bash
   git clone https://github.com maryammehmood202/Royal-cafe-.git
   ```

2. Move the project folder to your XAMPP `htdocs` directory.

3. Start Apache and MySQL using XAMPP.

4. Create a MySQL database.

5. Import the provided SQL file into phpMyAdmin.

6. Open your browser and visit:

   ```
   http://localhost/Royal-Cafe-Management-System
   ```
